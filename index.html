Help

1.) Draw nodes by clicking and dragging mouse

2.) Remove walls by holding shift and dragging (while clicking) the mouse

3.) Select a node type by clicking a radio button in the top menu

4.) Select an algorithm to use by clicking the dropdown menu in the top menu

5.) Press the start button to run the visualization. There is a play/pause button in the bottom menu

6.) For instantaneous path finding, click the instant path button in the bottom menu

If you have any questions, raise an issue on Github https://github.com/bscuron19/PathVis/issues
